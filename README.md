# dotfiles

- Window manager: `i3`
- Status bar: `polybar`
- File manager: `lf`
- `.gitconfig`
  - proxy
  - user info
- `.Xresources`
  - HiDPI
  - cursor theme & size
- `.xinitrc`: launch i3
  - HiDPI
  - fcitx
  - mouse & trackpad speed
  - autorandr
- `.config/autorandr`: multiple displays
- `.config/gtk-3.0` & `.config/gtk-4.0`
  - cursor theme & size
- `.config/chromium-flags.conf`
  - proxy
- `.config/logind.conf`: systemd logind
  - powerkey action
- `.ssh/config`
  - ssh via proxy
